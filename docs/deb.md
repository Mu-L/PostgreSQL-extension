
# DEB Extension Packages

There are **326** extensions available on Debian/Ubuntu compatible systems, **19** of them are Debian exclusive, and missing **19** EL exclusive extensions.

There are **70** built-in [**contrib**](contrib) extensions, in addition to **109** deb extensions provided by PGDG YUM repository, and **143** extensions provided by Pigsty.

There are **322** extensions available in the current major version PostgreSQL 16, and **290** ready for the latest PostgreSQL 17.
