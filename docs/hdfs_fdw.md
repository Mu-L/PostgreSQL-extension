```bash
pig ext add hdfs_fdw
```


Install `hdfs_fdw` via [Pigsty](https://pigsty.io/docs/pgext/usage/install/) playbook:

```bash
./pgsql.yml -t pg_extension -e '{"pg_extensions": ["hdfs_fdw"]}'
```
