Install `pg_walinspect` via the [`pig`](https://github.com/pgsty/pig) cli tool:

```bash
pig ext add pg_walinspect
```


Install `pg_walinspect` via [Pigsty](https://pigsty.io/docs/pgext/usage/install/) playbook:

```bash
./pgsql.yml -t pg_extension -e '{"pg_extensions": ["pg_walinspect"]}'
```


Install `pg_walinspect` [RPM](/rpm) from the **<span class="tcblue">CONTRIB</span>** **YUM** repo:

```bash
dnf install postgresql17-contrib;
dnf install postgresql16-contrib;
dnf install postgresql15-contrib;
```

