Install `random` via the [`pig`](https://github.com/pgsty/pig) cli tool:

```bash
pig ext add random
```


Install `pg_random` via [Pigsty](https://pigsty.io/docs/pgext/usage/install/) playbook:

```bash
./pgsql.yml -t pg_extension -e '{"pg_extensions": ["pg_random"]}'
```
