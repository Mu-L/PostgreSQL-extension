Install `topn` via [Pigsty](https://pigsty.io/docs/pgext/usage/install/) playbook:

```bash
./pgsql.yml -t pg_extension -e '{"pg_extensions": ["topn"]}'
```


Install `topn` [RPM](/rpm) from the **<span class="tccyan">PGDG</span>** **YUM** repo:

```bash
yum install topn_17*;

yum install topn_15*;
yum install topn_14*;
yum install topn_13*;
yum install topn_12*;
```
